MVC

		new beeMVC.Controller(function (models) {
		    var n=999;
		    function sub() {
		        n--;
		        models.time.set(n);
		    }
		    sub();
		    setInterval(sub, 1000);
		});
【代码说明】这里只操作数据模型，视图自动得到更新，操作视图的逻辑已经被封装了。
【MVC实现原理】核心理念是基于“观察者模式”实现，数据模型用来实例化一个数据对象，当数据发生改变的时候（调用set方法），就会被观察者捕获，并把消息及时发型给视图，从而实现模型改变，视图就跟着变的功能。控制器是一个粘合剂，在这里用来操作模型的相关逻辑。
